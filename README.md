# Susanna Maila Portfolio

A personal portfolio website showcasing the paintings and exhibitions of **Susanna Maila**, built with **React**, **React Router**, and **Contentful** for backend content management.

**Live Demo:** [https://susannamaila.com/](https://susannamaila.com/)

---

## Features

- **Dynamic Content from Contentful**
  - All pages, navigation menu, images, and external links are managed via Contentful CMS.
  - Supports rich text content, images, and videos.

- **Responsive Portfolio**
  - Optimized for desktop, tablet, and mobile.
  - Flexible image and video display with pop-up viewing.

- **Rich Text Rendering**
  - Uses Contentfulâ€™s rich-text renderer for paragraphs, headings, and hyperlinks.
  - Preserves formatting and displays external links with styling.

- **Navigation Menu**
  - Dynamic menu fetched from Contentful.
  - Highlights active page and supports multi-page portfolio navigation.

- **SEO Friendly**
  - Meta tags for title, description, keywords, and Open Graph.
  - Dynamically updates based on page content.

---

## Usage

1. Explore artworks, exhibitions, and artist pages.
2. Click images or videos to view in full.
3. Follow external links to exhibitions, related texts, or media content.
4. Navigation menu updates dynamically via Contentful CMS.

---

## Tech Stack

- **Frontend:** React, React Router, Tailwind CSS  
- **Backend:** Contentful CMS (content management for pages, images, and external links)  
- **Rich Text:** `@contentful/rich-text-react-renderer` for rendering structured text
